## About

This is a single-file HTML portfolio showcasing 20+ years of enterprise architecture, cloud transformation, and AI implementation experience. Built without frameworks — pure HTML, CSS, and JavaScript.

**Features**
- Light / dark mode with system preference detection
- Scroll-reveal animations
- Fully responsive (mobile, tablet, desktop)
- Self-contained — no external dependencies except Google Fonts
